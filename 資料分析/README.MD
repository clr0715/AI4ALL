## 學習作業:資料分析與descriptive statics(描述統計學)
```

描述統計學descriptive statics

推斷統計學（或稱統計推論，英語：statistical inference）
      統計學中，研究如何根據樣本數據去推斷總體數量特徵的方法。它是在對樣本數據進行描述的基礎上，
      對統計總體的未知數量特徵做出以概率形式表述的推斷。
      更概括地說，是在一段有限的時間內，通過對一個隨機過程的觀察來進行推斷的。
statistical hypothesis testing (confirmatory data analysis)

exploratory data analysis (EDA)
    https://en.wikipedia.org/wiki/Exploratory_data_analysis
```
